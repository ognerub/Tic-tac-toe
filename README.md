Mини-приложение "Крестики-нолики" (Tic-tac-toe game)

Описание: реализована возможность создания или выбора существующего игрока, сохранение между сессиями, вывод результатов игры, отправка этих данных на сервер (как пример), возможность выбора языка приложения без перезагрузки.

Инструкция по установке: скачать и запустить проект в Xcode

Системные требования: iOS не ниже версии 17.0

Стек: #Swift 6, #strict concurrency, #SwiftUI, #MVVM, #SwiftData, #User Defaults, #NavigationSplitView, REST API
